# 肝病文献统计方法调研报告

**日期：** 2026-07-21
**作者：** Eric Feng
**用途：** 开发团队内部参考，了解肝病领域论文中常用统计方法的分布

---

## 一、背景与目的

为了解肝病领域学术论文中实际使用的医学统计方法，以便在产品开发中对接临床研究规范，我们对 PubMed 上的肝病相关文献进行了自动化摘要分析，统计各类统计方法的出现频率。

---

## 二、数据来源与采集方法

**数据库：** PubMed（NCBI Entrez API，免费，无需 API Key）

**时间范围：** 2018–2024 年

**搜索词（共 6 组）：**

| # | 搜索词 |
|---|--------|
| 1 | hepatic fibrosis statistics |
| 2 | liver cancer statistical methods |
| 3 | hepatocellular carcinoma statistical |
| 4 | cirrhosis survival analysis |
| 5 | NAFLD statistical |
| 6 | liver function statistical methods |

**采集流程：**
1. 调用 `esearch` 接口，每组最多返回 150 篇，6 组合并去重
2. 调用 `efetch` 接口，100 篇一批，获取 XML 格式摘要（`<AbstractText>` 字段）
3. 对每篇摘要执行 42 个正则表达式，命中即计为该方法出现 1 次（同一篇不重复计）

**语料统计：**

| 指标 | 数值 |
|------|------|
| 检索到的唯一论文数 | 695 篇 |
| 成功解析摘要数 | 692 篇 |
| 摘要为英文的比例 | ~97%（抽样验证） |
| 与肝病相关的比例 | ~98%（抽样验证） |

**采集脚本：** `NLP/scripts/pubmed_stat_methods.py`（纯 Python stdlib，无需第三方依赖）

## 四、方法分类（按用途）

### 4.1 回归与预测建模（核心）
Cox 生存分析、Logistic 回归、线性回归、多变量分析、单变量分析、Lasso、列线图、竞争风险模型、限制性立方样条、混合效应模型

### 4.2 模型评估
ROC/AUC、C-statistic、校准曲线、决策曲线分析、DeLong 检验、交叉验证、灵敏度/特异度

### 4.3 因果推断
倾向评分匹配/IPTW、孟德尔随机化、中介分析、逆方差加权

### 4.4 统计检验（假设检验）
t 检验、卡方检验、Mann-Whitney/Wilcoxon、Fisher 精确检验、ANOVA、Kruskal-Wallis

### 4.5 数据质量与报告规范
置信区间、比值比、风险比、描述性统计、亚组分析、敏感性分析、多重插补

### 4.6 机器学习 / AI

### 4.7 合并分析
Meta 分析、逆方差加权、剂量-反应分析

## 五、主要发现

1. **生存分析是肝病研究的核心范式**：Cox 模型（14.7%）、竞争风险（1.3%）、HR（3.9%）加在一起，超过 20% 的论文涉及时间-事件分析，符合肝癌/肝硬化预后研究为主的特点。

2. **诊断模型评估成体系**：ROC/AUC（12.3%）→ C-statistic（3.3%）→ 校准曲线（1.4%）→ 决策曲线（1.3%），这四步是诊断/预测模型的标准评估流程，在论文里成套出现。

3. **因果推断工具兴起**：PSM/IPTW（3.8%）、孟德尔随机化（2.2%）排名进入前 20，说明观察性研究的方法学质量要求在提高。

4. **机器学习渗透但未主导**：随机森林+深度学习合计约 5% 的论文，ML 仍是补充手段，传统统计方法依然主流。

5. **CI/OR/HR 虚高**：置信区间排第一（25.9%）是因为它是结果报告格式，并非独立的分析方法；OR 和 HR 同理，实际统计"方法"应以 Logistic 回归和 Cox 为准。

## 六、对产品 / 算法落地的建议

**优先集成（高频刚需）**
- 生存分析套件：Cox 比例风险模型 + Kaplan-Meier 曲线 + Log-rank 检验，是预后研究的绝对基础
- 多变量 Logistic 回归：覆盖绝大多数二分类结局的临床研究场景

**模型评估三件套（成体系出现，需打包提供）**
- ROC/AUC → 校准曲线（Hosmer-Lemeshow / 实测-预测图）→ 决策曲线分析（DCA）
- 三步缺一则论文级别的模型评估不完整，工具链应一次性覆盖

**数据预处理模块**
- 多重插补（MICE）：15 篇论文提及，缺失数据处理已成规范，需作为数据清洗的标准选项
- 倾向评分匹配 / IPTW：26 篇提及，观察性数据因果推断的主流工具

**注意方法间的包含关系**
- 逆方差加权（IVW）是 Meta 分析和孟德尔随机化共用的汇总计算步骤，三者计数存在重叠；产品侧不必单独开发，作为底层计算提供即可
- C-statistic 与 AUC 在二分类场景下数值等价，ROC 模块覆盖后 C-statistic 自动满足

## 七、局限性

- **仅基于摘要**：论文摘要受字数限制，作者通常只写最核心的统计方法；t 检验、卡方检验、Mann-Whitney 等基础假设检验多出现在正文 Methods 部分而非摘要，本报告数据**显著低估**了这类方法在论文正文中的真实使用率
- **语言覆盖**：PubMed 以英文为主，中文期刊收录少，中文论文中的统计方法未覆盖
- **匹配方式**：正则关键词匹配，存在同义词漏网（如 "proportional hazards model" 未写 "cox" 时会漏）；部分方法存在包含关系（IVW 同时属于 Meta 分析和孟德尔随机化），计数存在重叠
- **计数单位**：每篇论文每种方法只计 1 次，不反映方法在单篇内的使用深度
- **搜索词偏差**：搜索词本身含 "statistical" 等词，可能偏向方法学描述更丰富的论文
